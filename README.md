# Dashboard de Seguridad · Bogotá

Tablero interactivo de hurtos y Línea 123 en Bogotá, hecho en una sola página HTML.

## Ver online

https://Santoo09.github.io/Proyecto-VDD/

## Ejecutar localmente

Abre `index.html` en el navegador. Requiere internet (usa Chart.js y Google Fonts desde CDN).

## Tecnologías

HTML, CSS, JavaScript y Chart.js.
